# Supermarket Management System

A supermarket database application built using **Python**, **Flask**, and **MySQL**.

## Overview

This project is a sales and warehouse management system for a supermarket.  
It allows customers to browse products, register, log in, add items to the cart, and complete purchases.  
It also provides an admin panel for managing products, categories, warehouses, employees, customers, and transactions. 

## Features

- Customer registration and login
- Product browsing with search and category filtering
- Shopping cart and checkout
- Order history and customer profile management
- Admin dashboard with sales statistics
- Product and category management
- Warehouse and stock management
- Employee and customer management
- Transaction tracking and order details

## Technologies Used

- Python
- Flask
- MySQL
- HTML / CSS

## Purpose

This project was developed for the **COMP333 Database Systems** course as a database application for a sales system that includes purchases, sales, and warehouses.
